# Projeto Doenças Cardiovasculares - EBAC

## 🎯 Objetivo
Este projeto faz parte do curso de Data Science da EBAC e tem como objetivo construir um modelo preditivo que auxilie na identificação de doenças cardiovasculares utilizando técnicas de Machine Learning e dados reais de saúde.

## 🛠️ Tecnologias Utilizadas
- Python 3.9+
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 📊 Metodologia
1. **Exploração dos dados**: análise inicial da base, verificação de tipos e identificação de padrões demográficos.
2. **Tratamento de dados**: limpeza, verificação de valores nulos e preparação das variáveis de entrada.
3. **Engenharia de Recursos**: processamento das variáveis (como idade, gênero, altura, peso, glicose, hábitos de fumo e consumo de álcool).
4. **Modelagem**: aplicação do algoritmo de Regressão Logística para classificação.
5. **Validação**: avaliação do modelo para garantir a correta identificação dos pacientes com risco cardiovascular.
